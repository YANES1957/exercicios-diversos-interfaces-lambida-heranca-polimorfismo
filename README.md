## Explicação Geral

Este repositório foi organizado para facilitar o estudo de **Programação Orientada a Objetos (POO)** em Java.  
Cada pasta contém exercícios completos e separados por tema, permitindo prática de:

- **Herança**: reutilização de código e criação de hierarquias de classes.  
- **Polimorfismo**: métodos que se comportam de forma diferente dependendo do objeto.  
- **Interfaces**: definição de contratos que classes podem implementar.  
- **Encapsulamento e Keywords**: controle de acesso a atributos e métodos, boas práticas de Java.  

Os desenhos em ASCII ajudam a visualizar a estrutura de classes e objetos de cada exercício.  
Este documento pode ser usado como guia de referência rápida para todo o conteúdo do repositório.
